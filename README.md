# Ruby Passenger Standalone

This is a Docker images for Ruby web apps.

## Configuration

These are the environment variables you can change:

- PORT: Passenger will bind to this port
- MIN_INSTANCES: Set passenger min instances
- MAX_POOL_SIZE: Set passenger max pool size
